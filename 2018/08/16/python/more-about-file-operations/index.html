StringIO/BytesIO

有时候读取不一定要从文件中，还可以从内存中，StringIO可以让你生成一个字符串，然后获取，BytesIO主要就是用于字节。注意的是，只有在声明时做了赋值，才可以通过类似于操作文件的read方法等进行读取，否则只能用getvalue。
